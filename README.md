## Setup Instructions

Follow these steps to run the project locally:

**1. Download or Clone the Repository**

```bash
git clone https://git@github.com:avishekdas2018/html-resume-page-assignment.git
```

**2. Download or Clone the Repository**

```bash
cd html-resume-page-assignment
```

**3. Open the Website**

- Double-click index.html

  **OR**

- Open it in a browser: With Live Server

## Usage

- Open index.html in any modern web browser.

- Scroll through the resume sections.

- Edit the HTML content to customize:
  - Your name

  - Skills

  - Experience

  - Projects

  - Education

  - Contact details
